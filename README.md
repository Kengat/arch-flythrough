# Облёт архитектуры — сайт + QR

Статический сайт для GitHub Pages: одна страница с видео-плеером.
QR-код ведёт на адрес этого сайта.

## Как поменять видео

Открой `config.js`:

- **Файл в репозитории** — положи ролик в папку `video/`, впиши имя в `videoFile`,
  а `youtubeId` оставь пустым. ⚠️ Лимит GitHub — 100 МБ на файл.
- **YouTube** — впиши ID ролика (часть ссылки после `v=`) в `youtubeId`.
  Подходит для тяжёлого HD-видео, лимита размера нет.

После изменений:

```bash
git add -A
git commit -m "обновил видео"
git push
```

Через ~1 минуту сайт обновится. QR-код менять не нужно — ссылка та же.

## QR-код

Лежит в `qr.png` (и `qr.svg`). Распечатай или вставь куда нужно.
Если поменяется адрес сайта — перегенерируй: `python make_qr.py <url>`.
